# MP3 Player

A basic Android MP3 Player application built using Kotlin in Android Studio as part of coursework practice.

## Features

* Play and pause songs
* Next and previous controls
* Local audio playback using Android MediaPlayer
* Simple Android UI

## Tech Stack

* Kotlin
* Android Studio
* Android SDK
* MediaPlayer API

## Purpose

This project was built to practice:

* Android application development
* Kotlin basics
* MediaPlayer integration
* Activity lifecycle and event handling
* UI development using XML layouts

## Run Locally

1. Clone the repository

```bash id="5th4h6"
git clone https://github.com/dpsnk454/Mp3Player.git
```

2. Open in Android Studio

3. Sync Gradle and run the app on an emulator or Android device
